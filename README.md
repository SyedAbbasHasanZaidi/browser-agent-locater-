# Browser Agent Element Locator

A TypeScript tool that reliably finds and interacts with elements 
on web pages using a multi-strategy approach:
- Vision-based detection (Claude/GPT-4V)
- DOM parsing (CSS/XPath selectors)
- Accessibility tree semantic search
- Automatic recovery on failure

Tested on 5 real websites. Logs trajectories for ML training.
